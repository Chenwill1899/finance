# 美股智能分析 — 使用说明

## 🚀 推荐：一键网页 App（输入代码 → 自动分析）

`analyzer_app.py` 是一个跑在你电脑上的本地网页。**输入股票代码、点「分析」，它自动后台跑脚本、画图、算指标、给买卖意见**——不用手动跑命令、不用复制粘贴。

```bash
proxy_on                                          # 先开代理
pip3 install yfinance --break-system-packages     # 只装一次
python3 analyzer_app.py                            # 启动（浏览器会自动打开）
```

然后在打开的页面（http://localhost:8765）输入框里打 `NVDA, RDW, RKLB`，点「分析」即可。看板会显示：现价、评分、K线走势、相关新闻、SPY/QQQ 相对强弱、各项技术指标、以及买点/止损/目标价的操作参考。按 `Ctrl+C` 停止。

> 注意：`analyzer_app.py` 和 `stock_data_fetcher.py` 必须放在**同一个文件夹**里。

---

## 进阶：直接用脚本（命令行）

`stock_data_fetcher.py` 会拉取真实历史行情，**精确计算**技术指标（MA / MACD / RSI / 量能 / 乖离率），并给出 100 分制综合评分和买卖信号。它只负责「取数据 + 算指标」——你也可以把它输出的 JSON 发给我，我来帮你做更深入的解读（含消息面）。

## 一、准备环境（只需做一次）

打开终端，先开代理，再装依赖库：

```bash
proxy_on
pip3 install yfinance --break-system-packages
```

> 说明：当前主流程只面向美股，只需要 `yfinance`。

## 二、运行

进入脚本所在文件夹后执行：

```bash
proxy_on
python3 stock_data_fetcher.py --stocks "NVDA,RDW,PLTR,RKLB" --days 120 --extras
```

参数说明：

- `--stocks`：美股 ticker，逗号分隔，如 `NVDA,RDW,RKLB`
- `--days`：取多少个交易日的历史（默认 120，够算各种均线）
- `--extras`：可选，拉取财报日和免 Key 新闻；网页 App 默认开启
- `--news`：可选，使用 Tavily / SerpAPI（需要 API key），一般不用

## 三、把结果发给我

脚本会在终端打印一段 JSON。**全选复制，粘贴到我们的对话里**，我就能基于精算数据帮你出决策看板（含买点 / 止损 / 目标价），还会自动搜最新消息面。

如果 JSON 太长，也可以存成文件再发我：

```bash
python3 stock_data_fetcher.py --stocks "NVDA,RDW,PLTR,RKLB" --days 120 --extras > result.json
```

## 四、脚本算了哪些东西

| 指标 | 用途 |
|---|---|
| MA5/10/20/60 | 均线排列，判断多空趋势 |
| MACD | 金叉死叉、零轴上下 |
| RSI 6/12/24 | 超买超卖（>80 强制不给买入） |
| 量能 | 放量/缩量，识别缩量回调买点 |
| 乖离率 | 偏离均线程度（>5% 不追高） |
| 相对强弱 | 对比 SPY / QQQ，判断是否真正跑赢大盘和科技指数 |
| 综合评分 | 100 分制 → 强烈买入/买入/持有/观望/卖出 |

## 常见问题

- **`ModuleNotFoundError: yfinance`** → 没装库，回到第一步 `pip3 install yfinance --break-system-packages`
- **`Tunnel connection failed: 403`** → 没开代理，先跑 `proxy_on`
- **某只股票报错** → 检查是否为美股 ticker；网页 App 目前只接受 1–5 个英文字母
